# Visual Novel Sketch Sequence (7 Scenes)

Ultra-consistent storyboard sketches for a dark atmospheric visual novel:
- Black background
- White line-art base
- Selective color accents only on composition-critical elements (roughly 20–30% of highlighted line elements)

## Scenes
1. `scene-01-awakening.svg` — distant figure in corridor, first tension beat.
2. `scene-02-threshold.svg` — approach to the mirror gate.
3. `scene-03-contact.svg` — hand-to-surface contact, energy response.
4. `scene-04-fracture.svg` — mirrored world splitting into paths.
5. `scene-05-pursuit.svg` — chase through geometric ruins.
6. `scene-06-revelation.svg` — central memory chamber.
7. `scene-07-resolution.svg` — balanced final frame and symbolic closure.

All files are lightweight SVG for easy preview directly in GitHub.
