# Image Factory QA Checklist

Use this checklist for every asset before connecting it to `StoryBeat.imageUrl` or `StoryScene.imageUrl`.

## Hard Gates

Reject or queue revision if any item fails.

| Gate | Pass condition |
|---|---|
| No API provenance | Image was generated by Codex built-in image generation, not by project OpenAI API code. |
| Scene accuracy | The image matches the selected shot objective and source beat. |
| UI safe zone | Lower 22 percent is quiet enough for a VN dialogue box. |
| No readable text | Screens contain abstract or blurred UI only. |
| No branding | No real YouTube/OpenAI/Codex/company logos in the image. |
| Protagonist continuity | Anonymous tired operator, dark hoodie/silhouette, same apartment continuity. |
| Anatomy | Hands, eyes, shoulders, and posture have no obvious generation artifacts. |
| Style lock | Ashen psychological realism, graphite/cold beige palette, no anime/cartoon drift. |
| Safety | No gore, weapons, celebrity likeness, or accidental real-person identity. |

## Sequence Continuity Gates

Use these for any multi-frame sequence, especially `raven_sequence_v2`.

| Gate | Pass condition |
|---|---|
| Seeded geometry | Objects used in later frames are visible or clearly implied in earlier frames. |
| Same location | Wall, window, alley axis, streetlamp, and ground plane remain spatially coherent. |
| Physical motion | Character/object movement can happen in real space between frames. |
| Bridge frame | Middle frame contains a clear carry-over from the previous frame and setup for the next frame. |
| No surprise perch | A raven cannot land on a ledge/sill/block that did not already exist. |
| Scale stability | Raven, human silhouette, window, and ledge keep plausible relative scale. |

## Scoring Rubric

Score 0-5. A frame should not become final unless all core scores are 4 or higher.

| Metric | 0 | 3 | 5 |
|---|---|---|---|
| Scene accuracy | misses the shot | mostly correct | exactly supports the beat |
| Style consistency | wrong style | acceptable drift | locked to style bible |
| Emotional clarity | unreadable | generally readable | readable in 1-2 seconds |
| Production usability | unusable | usable after edits | final-ready |
| Composition | cluttered | workable | VN-safe and cinematic |
| Artifact control | severe defects | minor defects | clean |
